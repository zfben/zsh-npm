# zsh-npm

Personal zsh plugin for npm.

| Alias | Command
| --- | ---
| `n` | `noglob npm`
| `ni` | `noglob npm install`
| `nu` | `noglob npm update`
| `nr` | `noglob npm run`
| `ne` | `noglob npm exec`
| `nd` | `noglob npm dev`
| `nt` | `noglob npm test`
| `ns` | `noglob npm start`
| `nl` | `noglob npm lint`

## Install

Using [zpm](https://github.com/zpm-zsh/zpm)

Add `zpm load zfben/zsh-npm` into .zshrc
